Overview

Decorative Geometry is the final project of CS89/189.5 Computational Photography,2015 fall. This work is inspired by "Decorative Mosaic" described in a ACM Siggraph 2001.

The initial idea for this final project was to implement decorative mosaic.in which a novel method for simulating decorative tile mosaics was proposaled based on Voronoi Diagram. It addresses several probelms like making tight alignment and retaining clear edges by emphasizing user-defined edges. When analyzing the basis for the paper: Voronoi diagram, I discovered some interesting presentation which finally leads me to the achievement of this project. The beautifulness of Voronoi digrams on which my decorative geometry based is shown below.

Analysis

This project presents two types of decorative geometry generated based on Voronoi Diagram and Centroid Voronoi Diagram. Specific methods contain manipulating vertices of each cell. In the phase of visulization, OpenGL is used for convenience. There are a lot more possiblities of artistic creation based on Voronoi Diagram and Centroid Voronoi Diagram, e.g, creating image mosaics of modern fashion and more.
